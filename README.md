# T9 MVP
This MVP version of T9 feature. Enjoy :smile:
## How to use 
```
yarn install
yarn build
yarn start
```
and open http://localhost:3000/

To run development server(without backend):
```
yarn start:dev
```

To run test cases:
```
yarn test
yarn test --coverage
```
